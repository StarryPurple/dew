# 关键字参考

## 表达式关键字

| 关键字 | 描述 | 示例 |
|---------|-------------|---------|
| `fn` | Lambda / 函数表达式 | `fn (x: Int) { x + 1 }`, `fn { 42 }` |
| `def` | Let 绑定（变量定义） | `def x = 42; x + 1` |
| `if` | 条件表达式 | `if cond { 1 } else { 0 }` |
| `else` | `if` 的替代分支 | `if cond { a } else { b }` |
| `fix` | 不动点组合子（递归） | `fix f: Int -> Int { ... f ... }` |
| `dup` | 复制值（对 Box 进行深拷贝） | `dup y = x` |
| `match` | 对 ADT 进行模式匹配 | `match x { Some(v) => v, None => 0 }` |
| `type` | 定义代数数据类型 | `type Option = None \| Some(Int)` |
| `include` | 文件包含预处理器 | `include "stdlib/list/nth.dew"` |

## 资源关键字（仿射类型）

| 关键字 | 描述 | 示例 |
|---------|-------------|---------|
| `box` | 分配线性盒子（仿射资源） | `box(42)` |
| `unbox` | 释放盒子，消耗它 | `unbox(b)` |

## 列表关键字

| 关键字 | 描述 | 示例 |
|---------|-------------|---------|
| `nil` | 空列表 | `nil` |
| `cons` | 列表构造器（头部, 尾部）；尾部是惰性的 | `cons(1, cons(2, nil))` |
| `head` | 获取列表的第一个元素 | `head(xs)` |
| `tail` | 获取列表的剩余部分（强制惰性尾部） | `tail(xs)` |
| `isnil` | 测试列表是否为空 | `isnil(xs)` |
| `[a, b, c]` | 列表字面量简写 | `[1, 2, 3]` |

## 字面量关键字

| 关键字 | 类型 | 描述 |
|---------|------|-------------|
| `true` | Bool | 布尔真 |
| `false` | Bool | 布尔假 |
| `()` | Unit | Unit 值（空元组） |

## 类型关键字

| 关键字 | 含义 | 示例 |
|---------|---------|---------|
| `Int` | 64 位有符号整数 | `fn (x: Int) { ... }` |
| `Bool` | 布尔 | `fn (x: Bool) { ... }` |
| `Unit` | Unit / void 类型 | `fn (x: Unit) { ... }` |
| `Box(T)` | 线性盒子类型 | `fn (x: Box(Int)) { ... }` |
| `List(T)` | 惰性列表类型 | `fn (xs: List(Int)) { ... }` |
| `T -> U` | Normal 函数（可多次调用） | `fn (f: Int -> Int) { ... }` |
| `T -1-> U` | 仿射 FnOnce（捕获了仿射资源） | 仅显示，由编译器推断 |

## 用户定义类型

| 语法 | 描述 | 示例 |
|--------|-------------|---------|
| `type N = A \| B(T)` | 代数数据类型 | `type Option = None \| Some(Int)` |
| `Ctor(args)` | 构造器（大写字母开头） | `Some(42)`, `None` |

## 模式语法（在 match 中）

| 模式 | 描述 | 示例 |
|---------|-------------|---------|
| `Ctor(sub)` | 构造器模式 | `Some(v)` |
| `x` | 变量绑定 | `v` |
| `_` | 通配符 | `_` |

## 运算符

| 运算符 | 含义 | 优先级 |
|----------|---------|------------|
| `!` | 严格性覆盖 | 1（最紧） |
| `*`, `/` | 乘法, 除法 | 2 |
| `+`, `-` | 加法, 减法 | 3 |
| `<`, `>`, `==` | 比较 | 4 |
| `->` | 管道前向 | 5（最松） |

## 注释

| 风格 | 语法 | 示例 |
|-------|--------|---------|
| Shell | `# 文本` | `# 这是一条注释` |
| C++ | `// 文本` | `// 这是一条注释` |
| 文档 | `/// 文本` | `/// 文档注释` |

## 分隔符

| 符号 | 用途 |
|-------|---------|
| `{` `}` | 代码块（函数体、if 分支、match 体） |
| `(` `)` | 分组、函数参数、函数调用 |
| `[` `]` | 列表字面量 |
| `:` | 类型标注分隔符 |
| `;` | 绑定续行 |
| `,` | 元素分隔符（列表、构造器参数） |
| `\|` | 类型声明中的变体分隔符 |
| `=>` | Match 分支箭头 |

## 诊断码

### 错误（D-前缀）

| 码 | 描述 |
|------|-------------|
| D001 | 仿射资源在消耗后再次使用 |
| D004 | 分支仿射不匹配 |
| D005 | 对 FnOnce 闭包使用 `dup`（无法通过捕获的仿射资源进行深拷贝） |
| D006 | 类型不匹配 |
| D007 | 未绑定变量 |
| D008 | 无效的 main 函数 |

### 警告（W-前缀）

| 码 | 描述 |
|------|-------------|
| W001 | 未使用的仿射资源 |
| W002 | Thunk 从未被强制（潜在死代码） |
