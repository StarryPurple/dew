# 元组和数组

Dew 提供两种结构复合类型：**元组**（异构定长集合）和**数组**（同构定长集合）。

## 元组

元组将固定数量的可能不同类型值组合在一起：

```dew
def pair = (42, true);          // (Int, Bool)
def triple = (1, 2, 3);         // (Int, Int, Int)
```

### 元组类型

```dew
def pair: (Int, Bool) = (42, true);
```

### 限制

- **无单元元组**：括号中的单个元素仅为分组，例如 `(42)` 是 `Int`，不是元组
- **无单元素元组**：元组需要两个或更多元素

### 元组更新

元组不可变。通过位置索引创建修改后的副本：

```dew
def t2 = t { 0 = 10 };  // (10, 2, 3)
```

## 数组

数组是定长、同构集合。大小是编译期常量：

```dew
def digits = [1, 2, 3];   // Array(Int, 3)
def flags = [true, false];  // Array(Bool, 2)
```

### 数组类型

```
Array(Int, 3)      // 三元素整数数组
Array(Char, 5)     // 五元素字符数组
```

### 数组访问

```dew
def first = digits[0];   // 1
```

### 数组更新

```dew
def digits2 = digits { [0] = 99 };  // [99, 2, 3]
```

## 字符串

字符串字面量是字符数组的语法糖：

```dew
"hello"   // 等价于 ['h', 'e', 'l', 'l', 'o']
          // 类型：Array(Char, 5)
```

没有单独的 `String` 类型。转义序列：`\n`（换行）、`\t`（制表）、`\\`（反斜杠）、`\"`（双引号）。

### 字符字面量

```dew
'a', 'Z', 'λ', '\n'
```
